# Codex Report — RD-13.1B: authorized receiver order override

Date: 2026-07-11

## Scope

Handoff task B only. The patch replaces the authorized read-only receiver recap with the supplied editable surface, adds a session-only endpoint, passes the override into order and shipping-recipient data during `confirm.confirm`, and clears it at checkout success. It does not update the account telephone or a saved address.

Optional patronymic is preserved in `custom_field` and `shipping_custom_field` JSON as `rd13_receiver_middlename`: stock `oc_order` has no middlename column, so no DB migration is introduced.
