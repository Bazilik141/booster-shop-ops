### Step 1 ‚Äî implement 301s in `.htaccess`
- Back up first: `cp .htaccess .htaccess.bak-legacy404-20260702`.
- Insert the rules in `/home2/boosters/public_html/.htaccess` **after** the HTTPS/www canonical redirect block and **before** the OpenCart catch-all `RewriteRule ^([^?]*) index.php?_route_=$1 [L,QSA]`. Wrap in a clearly-commented block:
```apache
# BEGIN legacy-404-301 20260702
RewriteRule ^product/OP-JP-MIX-MBX$ /product/One-Piece-Mystery-Box [R=301,L]
# ... one line per VERIFIED pair ...
# END legacy-404-301 20260702
```
- Rules are case-sensitive; match the exact old strings from the 404 log. No trailing slash unless the old URL had one. Use root-relative targets.
- Do NOT use `Redirect` (mod_alias) mixed with the existing `RewriteRule` engine ‚Äî stay with `RewriteRule [R=301,L]` for consistency and ordering control.
