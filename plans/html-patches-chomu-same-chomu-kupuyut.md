## ID=58 — Hot Wind Arena BOX
### Дія: лише перейменувати один рядок

Знайти:
```html
<h2>Чому корейський бокс</h2>
```
Замінити на:
```html
<h2>Чому саме Hot Wind Arena Box</h2>
```

## ID=79 — Mega Dream EX BST
### Вставити «Чому саме» + «Чому купують»

**Де вставляти:** одразу після блоку `<ul>` з chase-картами та параграфу про unweighed, **перед** `<div class="bs-faq-accordion"...>`.

```html
<h2>Чому саме Mega Dream EX</h2>

<p>
  Mega Dream EX виділяється серед японських релізів Pokémon TCG одразу кількома
  причинами. По-перше, це <strong>High Class Pack</strong> — формат із підвищеним
  вмістом рідкісних карт і гарантованим Pokémon ex у кожному паку. По-друге,
  сет знаменує <strong>перше повернення Mega Evolution у сучасній ері</strong>
  Scarlet &amp; Violet: покемони, яких фанати не бачили у новому форматі роками,
  повертаються в одному релізі. По-третє, введена нова рідкість —
  <strong>Mega Attack Rare (MA)</strong> — повноформатні монохромні карти атаки
  з унікальним фойлом, яких немає в жодному іншому сеті. Все це разом формує
  сильний попит як серед колекціонерів, так і серед гравців.
</p>
```

## Примітки

- **IDs 79–82** потребують цих вставок **разом із патчем H3→H2** — тобто всі правки краще об'єднати в один Codex handoff.
- **ID=58** — лише одна заміна рядка, можна вручну через OpenCart Admin.
- **ID=66 / 73** — назви секцій нестандартні, але виправдані специфікою товарів. Не чіпати.
- Блок «Чому купують» для **боксів** (ID=80) відрізняється: «sealed box у заводській стрічці» замість «оригінальні японські бустери». Для всіх інших бустер-паків — єдиний стандартний блок.
